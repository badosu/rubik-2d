Rubik-2d
========

A simple 2-dimensional rubik cube solver, implemented in C and ncurses.

Install
-------

    git clone https://github.com/badosu/rubik-2d.git
    cd rubik-2d
    make compile

Usage
-----

Edit the `initial_state` file in the format (excluding the indentation spaces):

     12
    345
    678

Run `./rubik-2d`
