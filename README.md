Rubik-2d
========

A simple 2-dimensional rubik cube solver, implemented in C and ncurses.

Install
-------

    git clone https://github.com/badosu/rubik-2d.git
    cd rubik-2d
    make compile

Usage
-----

Edit `initial_state` following in the format:

     15
    342
    687

Edit `goal_state` following the format:

     12
    345
    678

Run `./rubik-2d`
